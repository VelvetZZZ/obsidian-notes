# 无监督学习 (Unsupervised Learning)

> **💡 核心概念 (Core Concept)**
> 无监督学习与监督学习最大的不同在于：给定的数据**只有输入特征 $x$，没有输出标签 $y$ (Data only comes with inputs $x$, but not output labels $y$)**。
> 算法的任务是：在这些<mark style="background: #D2B3FFA6;">未标记 (unlabeled)</mark> 的数据中<mark style="background: #FFF3A3A6;">自动寻找</mark>**内在结构 (structure)** 或有趣的模式。

## 监督学习 vs. 无监督学习

* **监督学习 (Supervised Learning)**：从带有“标准答案”的**标记数据 (labeled data)** 中学习。例如给定肿瘤大小和年龄，并且已知它们是良性还是恶性（有红叉和紫圈的标签），去拟合一条边界。
* **无监督学习 (Unsupervised Learning)**：在**未标记数据 (unlabeled data)** 中寻找有趣的现象。例如只给你一堆蓝色的三角形数据点，算法自动发现它们可以分为两组（<mark style="background: #FFB8EBA6;">聚类</mark>）。

---

## 无监督学习的三大主要类型

根据课程内容，无监督学习算法主要包含以下三种功能：

### 1. <mark style="background: #FF5582A6;">聚类算法 (Clustering)</mark>
* **定义**：将相似的数据点<mark style="background: #CACFD9A6;">分组聚集在一起 </mark>(Group similar data points together)。
* **典型应用案例**：
    * **Google News (新闻聚合)**：每天互联网上有海量新闻，算法会自动将报道同一事件（例如不同媒体都在报道的“日本动物园大熊猫生下双胞胎”）的文章聚集到同一个专题下。
    * **DNA 微阵列分析 (DNA Microarray)**：分析个体的基因表达数据，算法自动发现结构，将具有相似基因特征的人群划分为不同的类型 (Type 1, Type 2, Type 3 等)。
    * **客户群组划分 (Grouping Customers)**：根据用户的行为特征进行市场细分。例如 deeplearning.ai 社区将学习者聚类为：“提升技能和知识 (growing skills and knowledge)”、“紧跟AI前沿 (stay updated with AI)”和“发展职业生涯 (develop career)”三大群体。

### 2. <mark style="background: #FF5582A6;">降维 (Dimensionality Reduction)</mark>
* **定义**：使用更少的数字来压缩数据 (Compress data using fewer numbers)，同时尽量保留原数据的信息。

### 3. <mark style="background: #FF5582A6;">异常检测 (Anomaly Detection)</mark>
* **定义**：在数据集中找出不寻常的、异常的数据点 (Find unusual data points)。

## 随堂测验：判断学习类型

在实际应用中，判断一个问题属于哪种学习方式的关键在于**数据是否有标签**。以下是课程中的判断练习：

| 任务描述 | 学习类型 | 判断依据 |
| :--- | :--- | :--- |
| 给定带有垃圾邮件/非垃圾邮件标签的邮件，学习垃圾邮件过滤器 | ❌ 监督学习 | 数据含有明确的标签 (labeled) |
| 给定网络上的新闻文章，将关于同一事件的文章分组 | ✅ 无监督学习 | 数据无标签，寻找数据分组结构 |
| 给定客户数据库，自动发现市场细分并将客户分组 | ✅ 无监督学习 | 数据无标签，属于典型的聚类问题 |
| 给定确诊为糖尿病或非糖尿病的患者数据集，分类新患者 | ❌ 监督学习 | 数据含有明确的疾病诊断标签 |
