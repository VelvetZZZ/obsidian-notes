# 监督学习 (Supervised Learning)

> **💡 核心定义 (Core Concept)**
> 监督学习的核心在于**学习输入到输出的映射关系 (Learning input-output or x-to-y mappings)**。
> 它的基本逻辑是：给算法提供包含**“正确答案” (right answers)** 的数据集，让算法从中寻找规律并学会预测。

## 典型应用场景 (X $\rightarrow$ Y Mapping)

监督学习在现实世界中有非常广泛的应用。系统通过接收输入 (X)，来预测特定的输出 (Y)：

| 领域 (Application) | 输入 X (Input) | 输出 Y (Output) |
| :--- | :--- | :--- |
| **垃圾邮件过滤** | 电子邮件 (email) | 是否为垃圾邮件 (spam? 0/1) |
| **语音识别** | 音频 (audio) | 文本转录 (text transcripts) |
| **机器翻译** | 英文 (English) | 西班牙文 (Spanish) |
| **在线广告** | 广告与用户信息 (ad, user info) | 是否点击 (click? 0/1) |
| **自动驾驶汽车** | 图像、雷达信息 (image, radar info) | 其他车辆的位置 (position of other cars) |
| **视觉检测 (工业)** | 手机照片 (image of phone) | 是否有缺陷 (defect? 0/1) |

##  监督学习的两大核心分支

监督学习主要分为**回归 (Regression)** 和 **分类 (Classification)** 两大类。

### 1. <mark style="background: #FF5582A6;">回归问题 (Regression)</mark>
* **核心特征**：用于预测一个**数字 (predict a number)**，其可能的输出结果是<mark style="background: #FFF3A3A6;">**无限多 (infinitely many)** 的连续值</mark>。
* **经典案例：房价预测 (Housing price prediction)**
    * **场景**：根据房屋面积 (House size) 预测房屋<mark style="background: #FFB8EBA6;">价格</mark> (Price)。
    * **模型拟合**：我们可以用一条直线 (straight line) 或者一条曲线 (curve) 来拟合数据点。例如，给定一个 750 平方英尺的房子，模型可以根据拟合的线条预测出价格大约是 15万美元（直线拟合）或 20万美元（曲线拟合）。

### 2. <mark style="background: #ADCCFFA6;">分类问题 (Classification)</mark>
* **核心特征**：用于预测**类别 (predict categories)**，其可能的输出结果是一个<mark style="background: #FFF3A3A6;">**少量的、有限的集合 (small number of possible outputs)**</mark>。
* **经典案例：乳腺癌检测 (Breast cancer detection)**
    * **单特征分类**：仅根据肿瘤大小 (tumor size) 预测它是良性 (benign, 标记为0) 还是恶性 (malignant, 标记为1)。
    * **多类别预测**：分类的输出不仅限于两个（0或1），也可以是多个有限的类别，比如：良性、恶性类型1、恶性类型2。
    * **多特征分类 (Two or more inputs)**：在实际情况中，我们通常会引入多个输入变量。例如，在二维坐标系中，同时使用**患者年龄 (Age)** 和**肿瘤大小 (Tumor size)** 作为输入特征，算法会在两类数据点（圈圈代表良性，叉叉代表恶性）之间学习出一条<mark style="background: #FFB8EBA6;">边界 (boundary) </mark>来进行分类。

## 核心对比总结

| 特性       | 回归 (Regression) | 分类 (Classification) |
| :------- | :-------------- | :------------------ |
| **预测目标** | 具体的数值 (Number)  | 特定的类别 (Categories)  |
| **输出结果** | 连续的、无限可能的输出     | 离散的、少量/有限的输出        |
| **典型例子** | 预测具体房价          | 判断肿瘤是良性还是恶性         |
