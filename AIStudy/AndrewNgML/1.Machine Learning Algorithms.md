# 机器学习算法总览 (Machine Learning Algorithms)

> **💡 核心思想 (The Big Picture)**
> 拥有算法只是一部分，更重要的是掌握**如何应用**这些算法（Practical advice for applying learning algorithms）。算法是我们解决真实世界问题的工具。

## 主要算法分类与课程路线图

本系列课程将重点介绍四大类机器学习算法，并分为三个子课程（Course 1, 2, 3）进行讲解：

### 1. 监督学习 (Supervised Learning)
* **课程分布**：将在 **Course 1** 和 **Course 2** 中深入探讨。
* **吴教授的行业洞见**：
    * **最广泛落地**：这是目前在现实世界商业应用中**使用最多**的算法 (Used most in real-world applications)。
    * **发展迅速**：该领域近年来取得了**极快的突破和进展** (Rapid advancements)。

### 2. 前沿与扩展算法 (Course 3)
在掌握监督学习后，**Course 3** 将涵盖另外三种不可或缺的算法方向：
* **无监督学习 (Unsupervised Learning)**：处理没有明确标签的数据，寻找数据内在的结构或模式。
* **推荐系统 (Recommender Systems)**：驱动现代互联网（如电商商品推荐、流媒体内容推送）的核心技术。
* **强化学习 (Reinforcement Learning)**：主要用于机器人控制、游戏AI等，通过“奖励与惩罚”机制让AI自主学习决策。
