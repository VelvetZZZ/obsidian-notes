## 0. 答疑：Math.ceil 是什么？(针对习题3)
在 Math 对象中，有三个“取整三剑客”，考试必考分辨：

| 方法 | 英文含义 | 作用 | 记忆口诀 | 例子 (2.1) | 例子 (2.9) |
| :--- | :--- | :--- | :--- | :--- | :--- |
| **`Math.floor()`** | Floor (地板) | **向下取整** | 往地上踩，去掉小数 | 2 | 2 |
| **`Math.ceil()`** | Ceiling (天花板) | **向上取整** | 往天上顶，有小数就+1 | 3 | 3 |
| **`Math.round()`** | Round (圆) | **四舍五入** | 经典的数学规则 | 2 | 3 |

> **习题3解析**：
> 题目要求 **0~7** 的整数。
> * `Math.random() * 8` 产生的最大数是 `7.999...`。
> * 用 `floor` (地板) 一压，就是 `7`。
> * 如果用 `ceil` (天花板)，`0.1` 也会变成 `1`，你就很难取到 `0` 了。所以生成随机整数通常用 **floor**。

```javascript
function 函数名(参数1, 参数2) {
```

## 2. 🔒 17.1.2 ECMAScript 闭包 (Closures) —— 难点

这是 JS 面试和考试中最抽象的概念。

- **定义**: **“定义在一个函数内部的函数”**。
    
- **核心能力**: 内部函数可以**访问**外部函数的变量，即使外部函数已经执行完了！
    
- **考试比喻**:
    
    - 外部函数是“老爸”，内部函数是“儿子”。
        
    - 老爸给儿子留了一笔钱（变量）。
        
    - 就算老爸退休了（函数执行结束），儿子依然可以用这笔钱。这就是闭包。
        
- **作用**: 保护变量不被全局污染，让变量常驻内存。

## 4. 📦 17.2 & 17.4 ECMAScript 对象 (重点中的重点)

JS 中万物皆对象。重点掌握以下 **4 大内置对象** 的常用方法。

### (1) String 对象 (字符串)

- `length`: 获取长度（属性，不是方法，没括号）。
    
- `charAt(index)`: 获取指定位置的字符（习题2考点）。
    
- `indexOf("a")`: 查找字符 "a" 第一次出现的位置（找不到返回 -1）。
    
- `substring(start, end)`: 截取字符串（含头不含尾）。
    
- `split(",")`: 把字符串切成数组（考点）。
    

### (2) Array 对象 (数组)

- `length`: 数组长度。
    
- `push()`: 在屁股后面**添加**元素。
    
- `pop()`: 在屁股后面**删除**元素。
    
- `join("-")`: 把数组拼成字符串（和 split 是反操作）。
    
- `reverse()`: 翻转数组。
    

### (3) Date 对象 (日期)

- **创建**: `var d = new Date();` (习题4考点)。
    
- **获取年份**: `d.getFullYear()` (注意不是 getYear)。
    
- **获取月份**: `d.getMonth()` ⚠️ **巨坑警告**：返回值是 **0~11** (0是1月，11是12月)。
    
- **获取星期**: `d.getDay()` (0是周日，1-6是周一到周六)。
    

### (4) Math 对象 (数学)

- **不需要 new**，直接用。
    
- `Math.random()`: 0~1 随机数。
    
- `Math.floor()` / `Math.ceil()`: 取整。
    
- `Math.max(a, b)` / `Math.min(a, b)`: 取最大/最小值。

## 5. 🛡️ 17.3 错误处理 (Error Handling)

当代码出错时，防止整个网页“挂掉”或白屏。

- **语法**

```JavaScript
try {
    // 可能会出错的代码
    // 比如：调用了一个不存在的函数
} catch (err) {
    // 如果上面出错了，就执行这里
    alert("出错了：" + err.message);
}
```

- **throw**: 主动抛出异常（自定义报错）。

## 第一部分：教材核心知识点精讲 (17.1 - 17.4)

### 1. 🧩 17.1.1 ECMAScript 函数 (Functions)
**考点**: 函数是 JS 的一等公民，也是代码复用的基础。
* **定义函数**:
    ```javascript
    function 函数名(参数1, 参数2) {
        // 函数体
        return 返回值; 
    }
    ```
* **arguments 对象 (重点)**:
    * 在函数内部自动存在的一个对象。
    * **作用**: 存储了传递给函数的所有参数（类似数组）。
    * **场景**: 用于参数个数不确定的情况。
    * *例*: `arguments[0]` 获取第一个参数，`arguments.length` 获取参数个数。

### 2. 🔒 17.1.2 ECMAScript 闭包 (Closures) —— 难点
**考点**: 能够解释什么是闭包，以及它的作用。
* **定义**: 闭包是指**有权访问另一个函数作用域中变量的函数**。简单说，就是“函数内部定义的函数”。
* **原理**: 内部函数引用了外部函数的变量，导致外部函数执行完毕后，其变量无法被垃圾回收机制回收，依然留存在内存中。
* **作用**:
    1.  **数据封装**: 模拟私有变量（不让外部随意修改）。
    2.  **保持状态**: 让变量的值在内存中持久化。

### 3. 🖱️ 17.1.3 HTML 事件 (Events)
**考点**: 事件三要素（事件源、事件类型、事件处理程序）。
* **事件处理程序的绑定方式**:
    1.  **HTML 内联**: `<button onclick="alert('Hi')">点我</button>`
    2.  **DOM 属性**: `document.getElementById("btn").onclick = function(){...}`
* **常用事件列表 (必背)**:
    * **鼠标**: `click` (点击), `mouseover` (移入), `mouseout` (移出).
    * **键盘**: `keydown` (按下), `keyup` (弹起).
    * **页面**: `load` (页面加载完成，初始化常用), `unload` (关闭页面).
    * **表单**: `focus` (获得焦点), `blur` (失去焦点 - 验证常用), `submit` (提交), `change` (内容改变).

### 4. 📦 17.2 ECMAScript 对象 (Objects)
**考点**: 一切皆对象。
* **对象的构成**: 由 **属性** (Properties) 和 **方法** (Methods) 组成。
    * 属性：对象的特征（如汽车的颜色）。
    * 方法：对象的行为（如汽车跑）。
* **创建自定义对象**: (见简答题详解)

### 5. 🛡️ 17.3 错误处理 (Error Handling)
**考点**: 如何防止程序崩溃。
* **try...catch 语句 (核心)**:
    ```javascript
    try {
        // 尝试执行的代码（可能会报错）
        var x = y + 1; // y未定义，会报错
    } catch (err) {
        // 捕获错误，处理善后，不让浏览器报错
        alert("发生错误：" + err.message);
    }
    ```
* **throw**: 主动抛出异常。 `throw "年龄必须大于0";`

### 6. 🛠️ 17.4 内置对象与本地对象 (核心考点)
**考点**: 熟练使用 String, Date, Math, Array 的 API。

#### (1) String 对象 (字符串)
* `length`: 长度属性。
* `charAt(index)`: 返回指定位置字符。
* `indexOf(str)`: 查找子串位置（找不到返回 -1）。
* `substring(start, end)`: 截取字符串。
* `split(separator)`: **分割字符串为数组** (如 `str.split(",")`)。

#### (2) Array 对象 (数组)
* **特点**: 长度动态、类型不限。
* `push()`: 末尾添加。
* `pop()`: 末尾删除。
* `join(separator)`: **数组元素连接成字符串** (如 `arr.join("-")`)。
* `reverse()`: 颠倒数组。
* `sort()`: 排序。

#### (3) Date 对象 (日期)
* `new Date()`: 获取当前时间。
* `getFullYear()`: 年。
* `getMonth()`: 月 (**0-11**，考点！)。
* `getDate()`: 日。
* `getDay()`: 星期几 (**0是周日**)。

#### (4) Math 对象 (数学)
* **特点**: 不需要 new，直接用 `Math.xxx`。
* `Math.random()`: 0-1 随机数。
* `Math.floor(x)`: 向下取整。
* `Math.ceil(x)`: 向上取整。
* `Math.round(x)`: 四舍五入。
* `Math.max(x,y)` / `Math.min(x,y)`: 最值。

## 第二部分：简答题标准答案 (P374 习题)

### (1) 举例说明如何用 JavaScript 创建自定义对象？
**答**: 主要有以下两种常用方法：
1.  **使用对象字面量 (最简单)**:
    ```javascript
    var student = {
        name: "张三",
        age: 18,
        study: function() { alert("正在学习"); }
    };
    ```
2.  **使用构造函数 (new Object)**:
    ```javascript
    var teacher = new Object();
    teacher.name = "李老师";
    teacher.teach = function() { alert("正在上课"); };
    ```
*(注：还可以写自定义构造函数 function Person(){}，但上面两种最基础常考)*

### (2) JavaScript 预定义的常用内置对象有哪些？
**答**: JS 提供了丰富的内置对象，常用的包括：
1.  **String**: 用于处理文本字符串。
2.  **Array**: 用于处理数组。
3.  **Date**: 用于处理日期和时间。
4.  **Math**: 用于执行数学任务（如随机数、取整）。
5.  **Boolean**: 布尔对象。
6.  **Number**: 数字对象。
7.  **RegExp**: 正则表达式对象。

### (3) 如何获取当前时间？
**答**: 使用 `Date` 对象。
1.  首先实例化 Date 对象：`var now = new Date();`
2.  然后调用其方法获取具体信息，例如：
    * `now.getFullYear()` (年)
    * `now.getMonth() + 1` (月，注意要加1)
    * `now.getDate()` (日)
    * `now.getHours()` / `getMinutes()` / `getSeconds()` (时分秒)

### (4) JavaScript 的数组有什么特色？如何定义和使用数组？
**答**:
* **特色**:
    1.  **类型宽松**: 数组元素可以是不同类型（如第1个是数字，第2个是字符串）。
    2.  **动态大小**: 不需要预定义长度，可以随时添加或删除元素，长度自动改变。
* **定义**:
    * `var arr = new Array();` (构造函数法)
    * `var arr = [1, 2, "a"];` (字面量法，推荐)
* **使用**:
    * 通过下标访问：`arr[0]`
    * 获取长度：`arr.length`

### (5) JavaScript 是如何处理事件的？JavaScript 中有哪些常用事件？
**答**:
* **处理机制**: JavaScript 采用**事件驱动**模型。当用户在网页上执行操作（如点击、输入）时，浏览器会触发相应的事件。通过编写**事件处理程序 (Event Handler)** 并将其绑定到特定元素上，就可以在事件发生时执行特定的代码。
* **常用事件**:
    * `onclick`: 鼠标点击。
    * `onload`: 页面或图片加载完成。
    * `onfocus` / `onblur`: 元素获得或失去焦点（常用语表单）。
    * `onchange`: 表单内容改变。
    * `onmouseover` / `onmouseout`: 鼠标移入或移出。
    * `onkeydown`: 键盘按下。
