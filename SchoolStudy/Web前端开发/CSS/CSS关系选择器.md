# CSS 关系选择器 (Relationship Selectors)
**日期**: 2026-01-10
**教材章节**: 第3章 CSS样式基础
**考点**: 符号含义、权重计算、选中范围

## 1. 核心关系图谱 (必背符号)

| 选择器名称 | 符号 | 语法示例 | 记忆口诀 | 选中范围 |
| :--- | :--- | :--- | :--- | :--- |
| **后代选择器** | **空格 ( )** | `div p` | 祖先找后代 | 选中 `div` 里面**所有的** `p` (包括儿子、孙子、重孙...) |
| **子代选择器** | **大于号 (>)** | `div > p` | 爸爸找亲儿子 | 只选中 `div` 的**第一级**子元素 (亲生儿子)，孙子不管 |
| **并集选择器** | **逗号 (,)** | `h1, p` | 我和你 | 选中 `h1` **和** `p` (群组打包，应用同一套样式) |
| **交集选择器** | **无符号** | `p.red` | 即是...又是... | 选中既是 `p` 标签，又拥有 `class="red"` 的那个元素 (标签名+类名) |
| **兄弟选择器** | **加号 (+)** | `h1 + p` | 紧挨着的弟弟 | 选中紧接在 `h1` 后面**第一个** `p` (必须相邻) |

## 2. ⚠️ 易错点解析

1.  **空格 vs 大于号**:
    * `div p` (后代): 范围大，株连九族。
    * `div > p` (子代): 范围小，只管一代。
2.  **交集选择器写法**:
    * **不能有空格**！ `div.box` (正确) vs `div .box` (错误，变成了<span style="background:rgba(240, 200, 0, 0.2)">后代选择器</span>)。
    * 标签名必须写在前面 (如 `p.red` 而不是 `.redp`)。
3.  **并集选择器**:
    * <span style="background:#fff88f">任何选择器都可以通过逗号组合</span>。
    * 例: `#header, .footer, p` (这三者都应用同样样式)。

## 3. 🎯 考试技巧
* **看图写代码**: 如果题目要求“将列表中的所有链接设为红色”，通常用后代 `ul li a { color: red; }`。
* **精准控制**: 如果题目强调“只修改第一层级”，一定要用 `>`。
