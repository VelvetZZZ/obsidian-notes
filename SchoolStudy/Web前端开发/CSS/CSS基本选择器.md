# CSS 基本选择器 (Basic Selectors)

**教材章节**: 第3章 CSS样式基础
**考点**: #符号区分  #唯一性原则 #优先级

## 1.  ID 选择器 (重点掌握)
* **定义**: 为页面中**唯一的**、具体的 HTML 元素指定样式。
* **语法符号**: **井号 (#)**
    * CSS 写法: `#id名 { 属性: 值; }`
    * HTML 写法: `<标签 id="id名">`
* **核心规则 (铁律)**:
    1.  **唯一性**:在一个 HTML 文档中，同一个 ID 名称（如 `header`）**只能出现一次**。它就像身份证号，不能两人共用。
    2.  **命名规范**: 不能以数字开头（`#2box` ❌），<span style="background:rgba(236, 57, 57, 0.55)">严格区分大小写</span>。
* **示例**:
 ```css
   /* 定义ID样式 */
   #green { color: green; }
```

```html
   <a href="#" id="green">我是绿色的链接</a>
```

## 2.  类选择器 (Class Selector)
* **定义**: 为页面中**某一类**标签指定统一的样式。
* **语法符号**: **点 (.)**
    * CSS 写法: `.类名 { ... }`
* **特点**:
    * **可复用**: 可以在多个标签上使用同一个 `class`。
    * **多类名**: 一个标签可以同时有多个类，<span style="background:rgba(240, 200, 0, 0.2)">用空格隔开</span>。
        * 例: `<p class="red font30">` (同时应用红色和30号字)。

## 3.  标签选择器 (Tag Selector)

* **定义**: 直接使用 HTML 标签名作为选择器。
* **作用**: 选中页面上**所有**该类型的标签（如让所有 `<a>` 去掉下划线）。
* **示例**: `a { text-decoration: none; }`。

## 4.  通配符选择器 (Universal Selector)

* **符号**: **星号 ( * )**
* **作用**: 匹配页面中**所有**元素。
* **常见用法**: <span style="background:rgba(240, 107, 5, 0.2)">清除默认边距</span>（Reset CSS）。
    ```css
    * { margin: 0; padding: 0; }
    ```
* **注意**: 即使不用，它也会遍历所有标签，降低网页执行速度，**不建议大量使用**。

### 教授的“重点解析” (Lecture Highlights)

关于你特别关心的 **ID 选择器**，我有两个考试技巧要传授给你：

1. **什么时候用 ID？**
    
    - 在考试的编程题中，通常只有**页面布局的大框架**（如网页头部 `#header`、侧边栏 `#sidebar`、底部 `#footer`）或者**特定的交互按钮**（如登录按钮 `#login-btn`）才会用 ID。
        
    - 其他的正文段落、列表项，一律优先用 Class。
        
    - **理由**：ID 权重太高（优先级高），一旦写死，后面想覆盖它很难。
    
2. **符号千万别搞混！**
    
    - 看到 `id="..."`，CSS 里一定要写 `#`。
        
    - 看到 `class="..."`，CSS 里一定要写 `.`。
        
    - 这是最“冤”的扣分项，阅卷老师一眼就能看出来。
