# CSS 伪类选择器 (Pseudo-class Selectors)
**教材章节**: 第3章 CSS样式基础
**考点**: #LVHA顺序原则 #冒号语法

## 1. 核心定义
* **定义**: 用于向某些选择器添加特殊的效果，这些效果无法通过简单的标签或类名来描述（例如鼠标是否悬停）。
* **语法符号**: **冒号 (:)**
    * 格式: `选择器:伪类 { 属性: 值; }`
    * 例: `a:hover { color: orange; }`

## 2. 🔗 链接伪类 (The "LVHA" Rule) —— ⭐⭐⭐⭐⭐ 必考死记
当我们要给 `<a>` <span style="background:rgba(240, 107, 5, 0.2)">（a 是元素选择器（标签选择器 / type selector），用来选中所有的超链接标签 ）</span>标签设置不同状态的样式时，**必须**严格按照以下顺序书写，否则部分样式会被覆盖而失效！

| 顺序 | 伪类写法 | 含义 | 记忆口诀 |
| :--- | :--- | :--- | :--- |
| **1** | `:link` | **未访问**的链接 (初始状态) | **L** (Lo) |
| **2** | `:visited` | **已访问**过的链接 (点过后) | **V** (Ve) |
| **3** | `:hover` | 鼠标**悬停**在元素上 (最常用！) | **H** (HA) |
| **4** | `:active` | 鼠标**按下未松开**时 (激活瞬间) | **A** (te) |

* **考试口诀**: **LoVe HAte** (爱恨原则) —— **L**ink, **V**isited, **H**over, **A**ctive。

## 3. 易错点 (Exam Trap)
1.  **顺序错误**: 如果把 `:hover` 写在 `:link` 前面，未访问的链接悬停时可能不会变色（因为 CSS 的<span style="background:#affad1">层叠性</span>，<span style="background:#d3f8b6">后面的规则覆盖前面的</span>）。
2.  **冒号是英文的**: `:hover` ✅ | `：hover` ❌ (中文冒号直接0分)。
3.  **适用范围**: 虽然叫链接伪类，但 `:hover` 和 `:active` 其实可以用于 `div`、`button` 等任何标签（比如鼠标移上去盒子变大）。


## 5. 🎨 常用文本样式技巧 (Text Tricks)
* **去除链接下划线 (必考)**:
    ```css
    a {
        text-decoration: none;
    }
    ```
* **应用场景**: 几乎所有的现代网页导航栏，链接默认都不带下划线，只有鼠标悬停 (`:hover`) 时可能加回来。
