## 1. 核心难点：相对路径 vs 绝对路径

在 HTML 中插入图片 (`<img src="...">`) 或超链接 (`<a href="...">`) 时，必须指定文件的位置。根据 PPT，这是**填空题**和**改错题**的重灾区。

### 1.1 相对路径 (Relative Path) ★★★
**定义：** 以**当前网页文件**所在的文件夹为起点，去寻找目标文件。
* **特点：** 只要整个文件夹移动，链接依然有效（网站内部引用推荐用这个）。

| 目标文件位置 | 写法规则 | 代码示例 (假设图片叫 `1.jpg`) |
| :--- | :--- | :--- |
| **同级目录** | 直接写文件名 | `src="1.jpg"` |
| **下级目录** (子文件夹) | `文件夹名/文件名` | `src="images/1.jpg"` (进入 images 文件夹找) |
| **上级目录** (父文件夹) | `../文件名` | `src="../1.jpg"` (**`../` 表示退回到上一层**) |
| **上上级目录** | `../../文件名` | `src="../../1.jpg"` (退回两层) |

> [!WARNING] 考试避坑
> 千万不要把斜杠写反了！HTML 路径通常用正斜杠 **`/`**，而不是 Windows 路径的反斜杠 `\`。

### 1.2 绝对路径 (Absolute Path)
**定义：** 带有完整地址信息的路径。
* **本地绝对路径：** 从盘符开始。
    * 例：`D:/web/images/1.jpg`
    * **缺点：** 换台电脑（由于盘符不同）可能就打不开了，**考试写代码极不推荐**。
* **网络绝对路径：** 带有协议的完整网址。
    * 例：`https://www.baidu.com/img/logo.png`
    * **用途：** 引用别人网站上的资源。

## 2. 文本修饰查漏补缺：下划线 `<u>`

我们在之前讲了加粗和斜体，还有一个标签用于给文字加下划线。

* **标签：** `<u>` (Underline)
* **语法：** `<u>需要加下划线的文字</u>`
* **注意：** 虽然 HTML5 依然支持，但 PPT 提示 HTML4.01 曾不推荐使用，建议用 CSS (`text-decoration: underline`) 代替。但在**期末考试 HTML 代码题**中，用 `<u>` 是最快最标准的写法。

## 3. HTML 特殊字符实体 (完整版)

键盘上的一些符号（如空格、`<`、`>`）在 HTML 代码中有特殊含义，不能直接显示，必须用“实体代码”代替。根据 PPT 和，请背诵下表：

| 显示符号 | 描述 | 实体名称 (★必背) | 记忆口诀 |
| :--- | :--- | :--- | :--- |
| **(空格)** | 空格 | **`&nbsp;`** | **N**on-**B**reaking **Sp**ace (牛B的空格) |
| **`<`** | 小于号 | **`&lt;`** | **L**ess **T**han |
| **`>`** | 大于号 | **`&gt;`** | **G**reater **T**han |
| **`&`** | 和号 | **`&amp;`** | **Amp**ersand |
| **`"`** | 双引号 | **`&quot;`** | **Quot**ation |
| **`©`** | 版权 | **`&copy;`** | **Copy**right |
| **`®`** | 注册商标 | **`&reg;`** | **Reg**istered |
| **`™`** | 商标 | **`&trade;`** | **Trade**mark |
| **`¥`** | 人民币/日元 | **`&yen;`** | **Yen** |

> [!TIP] 格式规范
> 所有的实体都必须以 **`&`** (and符) 开头，以 **`;`** (分号) 结尾。
> * 错：`&nbsp`
> * 对：`&nbsp;`

## 4. 👨‍🏫 教授的自测题 (Path & Entities)

1.  **填空题：**
    当前网页在 `web` 文件夹下，图片 `logo.png` 在 `web` 文件夹外面的上一层目录中。
    引用图片的正确代码是：`<img src="_________" />`
    * **答案：** `../logo.png`

2.  **代码阅读题：**
    如果网页上显示了 `Copyright © 2024`，对应的 HTML 源代码最有可能是？
    * A. `Copyright (c) 2024`
    * B. `Copyright &copy; 2024`
    * C. `Copyright <copy> 2024`
    * **答案：** **B**

3.  **判断题：**
    在 HTML 源代码中连续输入 5 个空格键，网页上也会显示 5 个空格的距离。
    * **答案：** ❌ **错！** (浏览器会将连续的空格合并为一个。想显示 5 个空格，必须写 5 个 `&nbsp;`)。
