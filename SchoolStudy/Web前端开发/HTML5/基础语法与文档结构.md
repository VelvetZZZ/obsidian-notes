## 1. HTML5 核心语法规范 (★ 考试手写代码必遵)

根据 PPT 和教材 P22，HTML 代码由 **标签 (Tag)** 和 **内容** 组成，必须遵守以下铁律：

### 1.1 标签的分类与写法
* **定义：** 由尖括号 `< >` 包围的关键词。
* **双标签 (成对出现)：**
    * **格式：** `<标签名> 内容 </标签名>`
    * **示例：** `<h1>这是一号标题</h1>`
    * **规则：** 必须有开始 `<...>` 和结束 `</...>`，否则会扣分。
* **单标签 (单独出现)：**
    * **定义：** 也就是空元素，没有内容的标签。
    * **格式：** `<标签名 />` (推荐) 或 `<标签名>`
    * **示例：** `<br />` (换行), `<hr />` (水平线), `<img ... />` (图片)。

### 1.2 嵌套规则 (重点)
* **树状结构：** 标签之间可以嵌套（父子关系），但**绝对不能交叉**！
    * ✅ **正确：** `<div> <p> 内容 </p> </div>`
    * ❌ **错误：** `<div> <p> 内容 </div> </p>` (这是“穿插”，浏览器解析会报错)。

## 2. 属性 (Attributes) 的使用规则

属性为标签提供附加信息。

* <span style="background:#d3f8b6"><font color="#4f6128">位置：</font> 只能写在 开始标签 的尖括号内部。</span>
* **格式：** <span style="background:rgba(236, 57, 57, 0.55)">属性名="属性值"</span>(Key-Value 对)。
* **多属性写法：**
    * 属性之间必须用 **空格** 隔开。
    * 顺序不限。
    ***示例：**
	      ```html
	       <a href="web/index.html" target="_blank">点击我吧</a>
	      ```
* **符号规范：** 所有的标点符号（引号、尖括号、等号）必须是 **英文半角符号**。

## 3. HTML5 标准文档结构 (背诵全文)

这是**填空题**和**改错题**的满分模板。根据教材 P21，一个标准的 HTML5 网页必须包含以下部分：

```html
<!DOCTYPE html>

<html lang="zh-CN"> 

    <head>
        <meta charset="utf-8"> <title>页面标题</title> </head>

    <body>
        这里写网页的主要内容...
    </body>

</html>
```
- **`<!DOCTYPE html>`**: 它不是 HTML 标签，而是声明指令，必须位于文档 **第一行**。

- **`<html>`**: 根元素，所有其他标签都在它里面。

- **`<body>`**: 定义文档的主体，其内容在浏览器窗口中可见。

- **文件扩展名**：建议使用 `.html`。

## 4.深入解析 `<head>` 头部标签 (★ 选择题高频考点)

`《head>` 是网页的“大脑”，虽然不显示内容，但对 SEO 和显示至关重要。

### 4.1 `<title>` 网页标题<span style="background:rgba(240, 107, 5, 0.2)">（必需的元素）</span>

- **作用：** 显示在浏览器 **选项卡 (Tab)** 的标题栏中。
    
- **位置：** 必须嵌套在 `<head>` 内部。
    
- **意义：** 搜索引擎判断网页内容的重要依据（通常是网站名称或文章题目）。


### 4.2 `<meta>` 元信息标签 (★ 难点)

`<meta>` 是单标签，用于定义页面的描述、关键词、作者等。 考试中喜欢考 `name` 属性和 `http-equiv` 属性的区别。

#### A. `name` 属性 (描述网页信息)

| **name 值**        | **含义** | **作用与示例 (★ 必背)**                                  |
| ----------------- | ------ | ------------------------------------------------- |
| **`keywords`**    | 关键字    | 告诉搜索引擎网页包含哪些关键词。<br><br>`content="HTML, CSS, JS"` |
| **`description`** | 网页说明   | 网页的概括性描述（摘要）。<br><br>`content="这是一个Web前端教程"`      |
| **`author`**      | 作者     | 标注网页制作者。<br><br>`content="ZhangSan"`              |
| **`generator`**   | 编辑软件   | 说明网站是用什么软件写的。<br><br>`content="HBuilder"`         |
#### B. `http-equiv` 属性 (模拟 HTTP 头部)

语法：`<meta http-equiv="参数名" content="参数值" />`。

| **http-equiv 值**   | **含义**   | **作用与示例 (★ 必背)**                                         |
| ------------------ | -------- | -------------------------------------------------------- |
| **`refresh`**      | **自动刷新** | 设置网页自动跳转或刷新。<br><br>`content="2;URL=http://..."` (2秒后跳转) |
| **`content-type`** | **编码格式** | (旧写法) 定义字符集。<br><br>`content="text/html;charset=utf-8"`  |
> [!TIP] HTML5 编码简写 在 HTML5 中，设置字符集通常使用更简洁的写法： `<meta charset="utf-8" />`


## 5. 常用全局属性 (Global Attributes)

根据教材 P43 表 2.3，这些属性几乎可以用在任何标签上：

| **属性名**               | **值**          | **描述/作用**                       |
| --------------------- | -------------- | ------------------------------- |
| **`id`**              | id值            | 定义元素的**唯一标识** (身份证号，页面内不能重复)。   |
| **`class`**           | 类名             | 定义元素的一个或多个**类名** (用于 CSS 样式分类)。 |
| **`style`**           | CSS样式          | 定义**行内 CSS 样式**。                |
| **`title`**           | 文本             | 定义**提示文本** (鼠标悬停时显示的小黄条)。       |
| **`lang`**            | 语言代码           | 定义元素内容的语言 (如 `zh` 中文, `en` 英文)。 |
| **`contenteditable`** | `true`/`false` | (HTML5新特性) 规定元素内容是否**可编辑**。     |
**问：** 标签名区分大小写吗？
- **答：** HTML5 <font color="#d83931">不区分大小写</font>（`<B>` 和 `<b>` 一样），但**W3C 建议所有标签名和属性名全部小写**。


## 6.  教授的考前避坑指南

1. **Meta 标签的混淆：**
    
    - 如果题目让你“设置网页关键词”，用 `name="keywords"`。
        
    - 如果题目让你“设置页面 3 秒后跳转”，用 `http-equiv="refresh"`。
        
    - 千万别把 `name` 和 `http-equiv` 搞混了！
        
2. **HTML 骨架：** 手写代码题第一步，先把 `<html><head><body>` 这个架子搭起来，不要上来就写内容，会扣结构分。
    
3. **标题位置：** `<title>` 标签只能放在 `<head>` 里，不能放在 `<body>` 里。
