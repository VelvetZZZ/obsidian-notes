## 1. 🚫 禁止写结束符 (Void Elements)

**语法规则：** 这些是空元素，绝对**不能**写 `</tag>`。
**常见错误：** 写了 `</br>` 或 `</input>` (扣分点)。

| **✅ 正确写法**       | 中文含义      | 📝 教授重点批注                 |
| :--------------- | :-------- | :------------------------ |
| **`base`, `br`** | 基准URL, 换行 | **❌ 严重错误修正**：这是两个标签粘在一起了！ |
| **`track`**      | 字幕轨道      | **❌ 大小写修正**：标签建议全小写。      |
| `img`            | 图片        | **★ 必考！** 千万别闭合。          |
| `input`          | 表单输入      | **★ 必考！** 表单核心。           |
| `hr`             | 水平线       | Horizontal Rule           |
| `link`           | 链接资源      | 引入 CSS 用。                 |
| `meta`           | 元信息       | 设置字符集 utf-8。              |
| `source`         | 媒体源       | 配合 video/audio 使用。        |
| `embed`          | 嵌入对象      | 万能插座。                     |
| `area`           | 热区        | 图像映射区域。                   |
| `col`            | 列属性       | 表格列设置。                    |
| `param`          | 参数        | 配合 object 使用。             |
| `keygen`         | 密钥生成      | *⚠️ 现代浏览器已废弃，但在教材里存在。*    |
| `command`        | 命令按钮      | *⚠️ 现代浏览器已废弃。*            |
| `wbr`            | 软换行       | 指定换行机会。                   |

## 2. 🔓 可省略结束符 (Optional Closing)

**语法规则：** 浏览器能自动补全闭合标签。
**🎓 考试建议：** 手写代码时**不要省略**，保持严谨习惯，防止阅卷老师误判。

| **✅ 正确写法**  | 中文含义  | 📝 教授重点批注                                |
| :---------- | :---- | :--------------------------------------- |
| **`thead`** | 表格头区域 | **❌ 严重拼写错误修正**：是 `thead` 不是 thread (线程)。 |
| `li`        | 列表项   | 用于 `<ul>` 或 `<ol>`。                      |
| `p`         | 段落    | 最常见的省略闭合标签。                              |
| `dt`, `dd`  | 定义列表项 | Definition Term/Description              |
| `tr`        | 表格行   | Table Row                                |
| `td`, `th`  | 单元格   | 数据/表头单元格。                                |
| `optgroup`  | 选项组   | 下拉菜单分组。                                  |
| `option`    | 选项    | 下拉菜单单项。                                  |
| `colgroup`  | 列组    | 表格列分组。                                   |
| `tbody`     | 表格主体  | -                                        |
| `rt`        | 注音    | 很少考。                                     |

---

## 3. 👻 可完全省略 (Omitted Tags)

**语法规则：** 即使代码里不写，DOM 树中也依然存在。
**🎓 考试建议：** 仅限**判断题**选“对”。手写代码必须写出完整结构。

* `html` (根元素)
* `head` (头部)
* `body` (主体)
* `tbody` (表格主体 - 浏览器会自动生成)

## 2. ⚡️ 考前自测：大家来找茬

**判断题：**
1. 在 HTML5 中，`<input>` 标签必须写成 `<input></input>`。
   * **答案：** ❌ (错！Input 是禁止写结束符的)
2. 在 HTML5 中，不写 `<body>` 标签，网页也能正常显示内容。
   * **答案：** ✅ (对！浏览器会自动补全 body，但考试写代码时请务必写上)
3. 代码 `<p>第一段 <p>第二段` 是合法的 HTML5 代码。
   * **答案：** ✅ (对！p 标签的结束符可以省略)
