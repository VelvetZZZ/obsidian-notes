# 核心考点：HTML图像标签  < img >

## 1. 基础语法与核心属性
**知识点来源**：示例 2-6-1
**标签性质**：单标签（Self-closing），不需要结束标签 `</img>`。
**基本语法**：
```html
<img src="图像路径" 属性名="属性值" ... />
```
关键属性表（必背）:

| **属性**     | **说明**                   | **教授划重点（考点）**                            |
| ---------- | ------------------------ | ---------------------------------------- |
| **src**    | **必须属性**。指定图像文件的路径（URL）。 | 如果路径错误或文件移动，图片将无法显示。考试常考相对路径写法。          |
| **alt**    | 替代文本。                    | **考题高频点**：当图像无法显示时（如路径错、网速慢），浏览器会显示此段文字。 |
| **width**  | 图像宽度。                    | 单位默认为 `px`（像素），也可以用百分比 `%`。              |
| **height** | 图像高度。                    | 如果只设置宽或高其中一个，另一个会根据**原比例自动缩放**（auto）。    |
| **border** | 图像边框宽度。                  | 单位是像素。设置 `0` 可取消边框。                      |
> [!WARNING] 注意事项 在网页插入图像时，一定要保证图像文件位于站点文件夹内。如果移动了文件位置但没修改 `src` 代码，图像将无法显示（显示为破碎图标）。

## 2. 图像与文本的对齐方式 (align)

**知识点来源**：示例 2-6-2 **属性作用**：`align` 属性用于定义图像与其**周围文本**的相对位置。

### 属性值详解

- **top**：将图像顶端与文本行最高的字符顶端对齐。
    
- **middle**：将图像的垂直中心点与文本的基线（baseline）对齐。（_注：PPT中描述为与中央对齐，考试以教材/PPT描述为准_）
    
- **bottom** (默认值)：将图像底端与文本基线对齐。
    
- **left**：**布局常用**。图像靠左，文本流向图像右侧（文字环绕效果）。
    
- **right**：图像靠右，文本流向图像左侧。
    

**语法示例**：

HTML

```
<img src="img/beijing.jpg" align="middle" />
```

## 3. 图像的间距设置 (hspace / vspace)

**知识点来源**：示例 2-6-3 **属性作用**：防止文本紧贴图像，通过增加留白提高可读性。

- **hspace** (Horizontal Space)：定义图像**左右两侧**的空白距离。
    
- **vspace** (Vertical Space)：定义图像**上下两端**的空白距离。

**语法示例**：

HTML

```
<img src="img/beijing.jpg" hspace="30" />
```

> [!TIP] 易错点提示 距离值是一个数值，**单位默认为 px（像素）**，在属性中可以省略单位不写。例如 `hspace="30"` 即代表 30px。

## 4. 【考试编程题模版】全属性综合写法

如果考试要求“**在页面中插入一张图片，宽度400，有3像素边框，与文字居中对齐，且左右间距30像素**”，请务必背诵以下代码结构：

HTML

```
<body>
    <img src="img/beijing.jpg" 
         width="400" 
         height="auto" 
         border="3" 
         alt="这是一张图像" 
         align="middle" 
         hspace="30" />
</body>
```
