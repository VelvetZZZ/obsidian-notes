# HTML5 超链接与锚点复习

## 1. 超链接基础 (`<a>` 标签)
**核心作用**：从一个网页指向另一个目标（网页、图片、文件、应用程序等）。
**特点**：
- 是**行内标签** (Inline Element)。
- 默认样式：蓝色字体，带下划线。
- 鼠标悬停时指针变为“小手”形状 👆。

### 1.1 基本语法
```html
<a href="链接地址">超链接内容（文字或图片）</a>
```
### 1.2 `href` 属性 (必选)

用于指定链接地址，取值分为：

- **绝对路径**：完整的 URL，如 `http://www.baidu.com`。
    
- **相对路径**：相对于当前文件的路径，如 `bigClass.html` 或 `images/photo.jpg`。
    

---

## 2. 打开方式 (`target` 属性)

**作用**：定义目标文档在何处显示。 **考点**：默认为 `_self`，常考 `_blank`。

| 属性值       | 描述                      | 备注                |
| --------- | ----------------------- | ----------------- |
| `_self`   | **默认值**。在当前窗口/标签页中打开。   | 如果没写 target，就是这个。 |
| `_blank`  | 在**新窗口**或新标签页中打开。       | **常用**，适合外部链接。    |
| `_parent` | 在当前框架的**父框架**中打开。       | 若无框架，等同于 `_self`。 |
| `_top`    | 在**整个浏览器窗口**中打开，清除所有框架。 | 用于跳出框架结构。         |

**代码示例**：

HTML

```
<a href="[http://www.baidu.com](http://www.baidu.com)" target="_blank">百度一下</a>
```

## 3. 锚点链接 (Anchor Links)

**作用**：实现页面**内部跳转**（如“回到顶部”、“跳转到目录”）。 **原理**：分为两步 —— “埋点” 和 “跳转”。

### 步骤 1：定义锚点 (埋点)

在页面的目标位置设置标记。

- **HTML5 推荐规范**：使用 `id` 属性。
    
- 兼容写法：使用 `name` 属性（旧版本，现在一般用 id）。
    

HTML

```
<a id="chapter1">第一章内容</a>
<h2 id="chapter1">第一章内容</h2>
```

### 步骤 2：链接到锚点 (跳转)

在 `href` 属性中使用 `#` 加上锚点名称。

HTML

```
<a href="#chapter1">跳转到第一章</a>
```

> **💡 教授的小提示 (Exam Tips)**：
> 
> 1. 锚点本身在页面上是不可见的（除非你给它加了样式），它就像一个隐形的书签。
>     
> 2. **手写代码题警示**：`href` 属性里千万别忘了那个井号 **`#`**！这是最容易扣分的地方。

### 👨‍🏫 教授的考前“划重点” (Exam Strategy)

针对你的考试题型，我有几个特别的嘱咐：

1.  **单选/多选题陷阱**：
    * 题目可能会问：“以下哪个属性值可以在新窗口打开链接？” 答案必须选 `_blank`。
    * 关于锚点，题目可能会问：“HTML5 规范建议使用哪个属性定义锚点？” 记得选 **`id`** 而不是 `name`（虽然 `name` 也能用，但 `id` 是标准）。

2.  **手写代码题技巧**：
    * **闭合标签**：`<a>` 是成对标签，写完 `</a>` 才能得分。
    * **属性引号**：虽然 HTML 容错率高，但考试时**属性值一定要加双引号**（如 `target="_blank"`），这体现了你的专业规范性。
    * **相对路径**：如果题目给的是同目录下的文件，直接写文件名即可；如果是下一级目录，记得写 `文件夹名/文件名`。

## 4. 图像链接 (Image Links)
**核心概念**：使用图片作为点击的“热区”来实现跳转。
**语法结构**：属于**嵌套结构**，即把 `<img>` 标签包裹在 `<a>` 标签内部。

### 4.1 基本语法
```html
<a href="目标链接地址">
    <img src="图片路径" alt="替换文本" />
</a>
```

### 4.2 属性详解

- **外层 `<a>`**：
    
    - `href`：决定点击图片后去哪里。
        
    - `target`：决定在哪里打开（如 `_blank`）。
        
- **内层 `<img>`**：
    
    - `src`：图片的来源路径（绝对或相对）。
        
    - `alt`：图片无法显示时的替代文本（**考试常考点：规范性**）。
        
    - `width` / `height`：控制图片大小。
        

**代码示例** (参考教材示例 2-7-4)：

HTML

```
<a href="[http://www.baidu.com](http://www.baidu.com)">
    <img src="img/logo.gif" alt="网站Logo" width="300" height="auto">
</a>
```

> **💡 教授的小提示 (Exam Tips)**：
> 
> 1. **视觉效果**：PPT 里特意提到，视觉上图像链接和普通图像没区别，唯一的暗示是鼠标悬停时指针会变成**手形** (👆)。
>     
> 2. **标签闭合**：手写代码时，很容易写完 `<img ...>` 就忘了写外面的 `</a>` 关闭标签，**千万别漏！** 这叫“有始有终”。
>     
> 3. **嵌套顺序**：一定是 `<a>` 包着 `<img>`，**不能反过来**。你想想，是“链接包含着图片”，而不是“图片肚子里有个链接”。
