#  期末重点速记：HTML5 无语义标签与特殊字符

**标签：** #Web基础 #HTML5 #布局 #块级与行内 #考前必背
**资料来源：** 重点PPT(无语义标签/div/span) & 教材对应章节

---

## 1. 核心概念：无语义标签 (Non-semantic Tags)

根据 PPT，HTML 中绝大多数标签（如 `h1`, `p`, `img`）都有明确语义，而 `<div>` 和 `<span>` 是两个特殊的**无语义标签**。

* **作用：** 它们仅仅是容器，用于**“分组”**或**“包裹”**内容，以便通过 CSS 进行样式设置或布局。

---

## 2. ☠️ 终极对决：`<div>` vs `<span>` (★ 必考辨析)

这是考试中**选择题**和**填空题**的高频考点，必须死记硬背它们的区别！

| 维度 | `<div>` (Division) | `<span>` |
| :--- | :--- | :--- |
| **PPT定义** | 用于**划分区域**，进行页面布局。 | 用于**修饰文本**，区别于周围文字。 |
| **元素类型** | **块标签 (Block Tag)** | **行标签 (Inline Tag)** |
| **换行行为** | **独占一行** (自动换行) | **不换行** (与文字在一行显示) |
| **能否包含** | 能包含其他块级元素（如 `h1`, `p`）和行内元素。 | 只能包含文本或其他行内元素。 |
| **记忆比喻** | 📦 **大纸箱** (装东西用的) | 🖍️ **荧光笔** (画重点用的) |

---

## 3. 代码实战示例

> [!EXAMPLE] 典型布局代码
> 观察以下代码，理解嵌套关系：`div` 是大容器，`span` 是小修饰。

```html
<body>
    <div>
        <h2>这是标题</h2>
        
        <p>这是一段普通的文本，但这里有 <span style="color:red">特殊</span> 的标记。</p>
    </div>
</body>
```
## 4. 补充考点：HTML 特殊字符实体 (Character Entities)

HTML 代码中不能直接写 `<` 或 `>`（会被误认为是标签），必须用“实体”代替。

| **显示符号**      | **实体代码 (Entity Name)** | **记忆方法**                      |
| ------------- | ---------------------- | ----------------------------- |
| **`<`** (小于号) | **`&lt;`**             | **L**ess **T**han             |
| **`>`** (大于号) | **`&gt;`**             | **G**reater **T**han          |
| **空格**        | **`&nbsp;`**           | Non-Breaking Space (PPT未提但常考) |
| **`&`** (和号)  | **`&amp;`**            | Ampersand                     |
>[!WARNING] 考试注意 实体代码必须以 **`&`** 开头，以 **`;`** 结尾！漏掉分号会扣分。

## 5. 教授的自测题

1. **判断题：** `<span>` 标签可以用来划分一个大的网页版块（如页脚）。
    
    - **答案：** ❌ (错！划分版块要用块级元素 `<div>` 或语义化标签 `<footer>`，`span` 太小了且不换行)。
        
2. **填空题：** 想要在网页上显示 `<h1>` 这个标签原本的字符（而不是把它解析成标题），应该写成：________ h1__ ___。
    
    - **答案：** `&lt;`  `&gt;`。


# HTML5 文本与段落排版

**标签：** #Web基础 #HTML5 #排版标签 #易错点 #考前必背
**资料来源：** 重点PPT(标题/段落/Font/修饰) & 教材对应章节

---

## 1. 块标签 (Block Tags)

这些标签会自动换行，独占一行。

### 1.1 标题标签 `<h1>` ~ `<h6>`
* **定义：** 用于定义文档的各级标题，`<h1>` 最大，`<h6>` 最小。
* **属性 (考试常考)：**
    * `align`: 设置对齐方式。值：`left` (默认), `center`, `right`。
* **🚨 教授排雷：**
    * HTML 中 **没有 `<h7>` 标签**！写了也不会报错，但只是普通文字。
    * 标题文字默认是**加粗**的。

### 1.2 段落标签 `<p>`
* **定义：** Paragraph。定义一个段落。
* **特征：** 段前和段后会有默认的**间距** (空行)。
* **属性：**
    * `align`: 对齐方式 (`left`, `center`, `right`)。

### 1.3 分隔线标签 `<hr />`
* **定义：** Horizontal Rule。在页面画一条水平线，用于分隔内容。
* **语法：** 单标签，推荐写法 `<hr />`。

---

## 2. 行标签 (Inline Tags)

这些标签不会换行，只修饰文字。

### 2.1 强制换行 `<br />`
* **定义：** Break。强制文本换行，但不会像 `<p>` 那样产生大间距。
* **语法：** 单标签 `<br />`。

### 2.2 字体标签 `<font>` (★ 古董)
虽然 HTML5 不推荐，但教材重点讲解了，**考试必考属性**！

* **语法：** `<font size="字号" color="颜色" face="字体">文本</font>`
* **三大属性：**
    1.  **`size`**: 字体大小。值为数字 **1~7** (注意不是像素px，3是默认大小)。
    2.  **`color`**: 字体颜色。可用英文单词 (`red`) 或 十六进制 (`#FF0000`)。
    3.  **`face`**: 字体名称。如 "宋体", "黑体"。

### 2.3 文本修饰：物理 vs 语义 (★ 辨析题重点)

考试中如果问“推荐使用哪个”，请优先选 **语义化标签**。

| 效果 | **物理标签** (只管样子) | **语义标签** (强调语气 - ★推荐) | 区别 (PPT考点) |
| :--- | :--- | :--- | :--- |
| **加粗** | **`<b>`** (Bold) | **`<strong>`** | `<strong>` 表示**语气加重**，搜索引擎更重视。 |
| *斜体* | **`<i>`** (Italic) | **`<em>`** (Emphasis) | `<em>` 表示**强调**，HTML5 规范建议优先使用。 |

---

## 3. 代码实战 (考试填空题模板)

> [!EXAMPLE] 综合排版示例
> 题目：将“期末复习”设置为标题居中；下一行显示“重点”两个字，要求：3号字、红色、宋体、加粗。

```html
<body>
    <h1 align="center">期末复习</h1>
    
    <p>
        <font size="3" color="red" face="宋体">
            <strong>重点</strong>
        </font>
    </p>
</body>
```
## 4. 教授的自测题

1. **判断题：** 在 HTML 中，`<h1>` 标签显示的文字一定比 `<h3>` 大。
    
    - **答案：** ✅ (在默认样式下是对的，PPT 图示也是如此)。
        
2. **单选题：** 下列哪个标签用于在网页中产生一个强制换行？
    
    - A. `<p>` B. `<br>` C. `<hr>` D. `<break>`
        
    - **答案：** **B** (`<p>`是段落，`<hr>`是线)。
        
3. **代码改错：** `<font size="12px">文字</font>`
    
    - **错误点：** `<font>` 的 `size` 属性值是 **1到7的数字**，不能加 `px` 单位！(这是个超级大坑)。
