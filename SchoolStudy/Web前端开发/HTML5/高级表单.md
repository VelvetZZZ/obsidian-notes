# HTML5 高级表单与框架 (Advanced Forms & Frames)

## 1. 多行文本域 (`<textarea>`)
**适用场景**：当用户需要输入大量文字时（如：留言板、评论、自我介绍）。
**特点**：不同于 `<input>` 是单标签，`<textarea>` 是**双标签**。

### 1.1 语法结构
```html
<textarea name="控件名称" rows="行数" cols="列数">
    这里是默认显示的文字（如果有的话）
</textarea>
```
### 1.2 关键属性

|**属性**|**说明**|**备注**|
|---|---|---|
|`rows`|规定文本区内的可见**行数** (高度)。||
|`cols`|规定文本区内的可见**宽度** (字符数)。||
|`name`|提交给服务器的变量名。|**必填**，否则无法提交数据。|

> [!TIP] **⚠️ 教授划重点 (Exam Tips)**：
> 
> - **默认值陷阱**：`<textarea>` **没有** `value` 属性！如果你想设置默认内容，必须写在**开始标签和结束标签之间**。
>     
> - _错误写法_：`<textarea value="默认文字" />`
>     
> - _正确写法_：`<textarea>默认文字</textarea>`
>

## 2. 下拉列表 (`<select>`)

适用场景：省份选择、日期年份选择等，节省页面空间。

结构：由父标签` <select> `和子标签 `<option>` 组成。

### 2.1 语法结构

```html
<select name="控件名称">
    <option value="提交的值1">显示给用户看的内容1</option>
    <option value="提交的值2" selected>显示给用户看的内容2 (默认选中)</option>
</select>
```

### 2.2 关键属性

- **`<select>` 属性**：
    
    - `name`：整个列表提交时的变量名。
        
- **`<option>` 属性**：
    
    - `value`：**实际提交给服务器的值**（用户看不见）。
        
    - `selected`：设置默认选中的项。
        
        - 写法：`selected="selected"` 或简写为 `selected`。

## 3. 浮动框架 (`<iframe>`)

**核心概念**：在当前网页中挖一个“洞”，在这个洞里加载并显示**另一个网页**。

### 3.1 语法结构

```html
<iframe src="默认加载的网址" name="框架名称"></iframe>
```

### 3.2 常用属性 (考点)

|**属性**|**说明**|**重点**|
|---|---|---|
|`src`|初始加载的页面地址。||
|**`name`**|框架的名称。|**最重要**！用于配合超链接的 `target` 属性。|
|`width`/`height`|框架的宽和高。||
|`frameborder`|边框宽度。|`0` 表示无边框。|
|`scrolling`|是否显示滚动条。|`yes`(常显), `no`(不显), `auto`(自动)。|

### 🔥 必考难点：Iframe 与 链接的联动

这是期末考试特别爱出的**编程题**模式：点击左边的链接，右边的框架内容发生变化。

**原理**：

1. 给 `<iframe>` 起个名字，例如 `name="myFrame"`。
    
2. 给 `<a>` 标签设置 `target` 属性，值等于那个名字，例如 `target="myFrame"`。
    

**代码示例** (参考 PPT 示例 2-10-1)：

```html
<a href="[http://www.baidu.com](http://www.baidu.com)" target="aa">百度</a>

<iframe name="aa" src="[http://hao.360.com](http://hao.360.com)" width="500" height="200"></iframe>
```

> [!TIP] 小提示：
> PPT 最后提到，在 HTML5 标准中，frameborder 和 scrolling 属性其实已经被废弃了，建议用 CSS 来控制。但在考试中，如果题目要求你“设置无边框”或“隐藏滚动条”，请依然优先使用 PPT 上教的这两个属性，以确保得分。
