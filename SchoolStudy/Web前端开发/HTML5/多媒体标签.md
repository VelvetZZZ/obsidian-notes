# 1. `<embed>`

**标签：** #Web基础 #HTML #多媒体 #简单记忆
**对应PPT：** 多媒体标签 `<embed>`

---

## 1. 核心标签：`<embed>` (嵌入)

这是一个**单标签** (空标签)，通常不需要闭合标签 `</embed>`，但在 XHTML 规范中建议写成 `/>` 结尾。

**功能：** 用于插入各种多媒体资源（视频、音频、动画）。

| 属性名 | 英文全称 | 作用 (★必考) | 示例值 |
| :--- | :--- | :--- | :--- |
| **`src`** | **S**ou**rc**e | **资源路径**。告诉浏览器文件在哪里。 | `"video.mp4"`, `"music.mp3"` |
| **`width`** | Width | 播放窗口的宽度 (px)。 | `"300"` |
| **`height`** | Height | 播放窗口的高度 (px)。 | `"200"` |

---

## 2. 代码示例 (背诵模板)

> [!EXAMPLE] 插入视频的标准写法
> 题目：请在网页中插入一个名为 `sea.mp4` 的视频，宽度300，高度200，文件存放在 `object` 文件夹下。

```html
<embed src="object/sea.mp4" width="300" height="200" />
```
## 3. 🕸️ 教授的“考古”知识点 (选择题/判断题)

PPT 中提到了两个额外的标签，考试可能会考辨析：

1. **`<bgsound>` (背景音乐)**
    
    - **考点：** 如果题目问“哪个标签用于设置网页的**背景音乐**”，首选这个。
        
    - **特性：** 它是旧版 Internet Explorer 的专用标签，现在的 Chrome/Firefox 其实不怎么支持它了（现在都用 `<audio>`），但**教材和考试里它依然是标准答案**。
        
2. **`<object>` (对象)**
    
    - **考点：** 功能与 `<embed>` 类似，有时题目会问“除了 embed 还有哪个标签能插视频”，选 `<object>`。
        

> [!TIP] HTML5 新标准 (加分项) 如果是在**编程题**中，题目没有强制要求用 `<embed>`，目前更推荐使用 HTML5 原生标签：
> 
> - 插入视频：`<video src="...">`
>     
> - 插入音频：`<audio src="...">`
>     
> 
> 但在填空题中，如果给出的空只有 `src`, `width`, `height` 且是单标签，优先填 `<embed>`。

# 2.  `<object>`

**标签：** #Web基础 #HTML #多媒体 #易错点
**对应PPT：** 多媒体标签 `<object>` 插入音频

---

## 1. 核心标签：`<object>` (对象)

功能与 `<embed>` 类似，用于在网页中加载多媒体文件（如音频、视频、Flash、PDF等）。

**⚠️ 关键语法 (背诵)：**
它是**双标签**，属性名是 **`data`**。

| 属性名 | 含义 | 作用 (★必考) | 记忆口诀 |
| :--- | :--- | :--- | :--- |
| **`data`** | 数据源 | **指定文件的路径**。<br>❌ 也就是 `<embed>` 里的 `src`，千万别写错！ | **Obj**ect 也就是 **D**ata (OD组合) |
| **`width`** | 宽度 | 播放器的宽度 (px)。 | - |
| **`height`** | 高度 | 播放器的高度 (px)。 | - |

---

## 2. ☠️ 死亡陷阱：Embed vs Object

这是考试中填空题和改错题的高频考点，请务必区分：

| 维度 | `<embed>` (嵌入) | `<object>` (对象) |
| :--- | :--- | :--- |
| **文件路径属性** | `src="..."` | `data="..."` (★最易错) |
| **标签类型** | 单标签 `<embed />` | 双标签 `<object>...</object>` |
| **兼容性 (教材)** | 通用性较好 | IE支持好，其他可能不支持 |

---

## 3. 代码示例 (满分模板)

> [!EXAMPLE] 插入音频的标准写法
> 题目：使用 `<object>` 标签插入 `object/bana.mp3`，宽400，高50。

```html
<object data="object/bana.mp3" width="400" height="50"></object>
```
# 3. 背景音乐标签 `<bgsound>`

**标签：** #Web基础 #HTML #多媒体 #古董标签 #IE专用
**对应PPT：** 背景音乐 `<bgsound>`

---

## 1. 核心标签：`<bgsound>`

这是一个**非标准**但教材中存在的标签，用于设置网页背景音乐。
* **位置：** 通常写在 `<head>` 或 `<body>` 标签内部。
* **特点：** 页面上**看不见**播放器，只有声音。

| 属性名 | 含义 | 作用 (★必考) | 特殊值记忆 |
| :--- | :--- | :--- | :--- |
| **`src`** | Source | 音乐文件的路径。 | `"music.mp3"` |
| **`loop`** | 循环次数 | 控制音乐播放多少遍。 | **`-1`** (代表**无限循环**) |

---

## 2. 代码示例 (背诵模板)

> [!EXAMPLE] 无限循环背景音乐
> 题目：设置网页背景音乐为 `object` 文件夹下的 `bana.mp3`，要求打开网页后无限循环播放。

```html
<bgsound src="object/bana.mp3" loop="-1" />
```

> [!WARNING] 兼容性警告 (判断题)
> 
> - **Q:** `<bgsound>` 可以在所有浏览器中运行吗？
>     
> - **A:** **不可以！** 它主要是 **IE 浏览器** 的专用标签。
>     
> - _在现代浏览器（Chrome, Edge, Firefox）中，这个标签已经被废弃。但在做期末考卷时，请尊重教材，认定它就是做背景音乐的方法。_
>     

> [!TIP] 关联记忆：三大多媒体标签
> 
> 1. **`<embed>`**: 通用嵌入 (src)
>     
> 2. **`<object>`**: 对象嵌入 (data)
>     
> 3. **`<bgsound>`**: 背景音乐 (loop=-1, IEonly)
