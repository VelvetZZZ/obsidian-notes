# 运算符与表达式 

## 1. 运算符全家桶 (Overview) 
根据 PPT 的思维导图，Java 运算符主要分为六大派系。期末考 **90%** 的分数集中在前三类：
1.  **赋值运算符** (`=`)
2.  **算术运算符** (`+`, `-`, `*`, `/`, `%`, `++`, `--`)
3.  **比较运算符** (`>`, `==`, `!=`)

## 2. 算术运算符的两个“隐藏坑” 

### 坑点 A：整数除法 (Integer Division)
> **铁律**：整数 / 整数 = 整数。小数部分会直接被**砍掉**，<span style="background:rgba(236, 57, 57, 0.55)">不是四舍五入</span>！

```java
int f = 9 / 2; 
// 结果是 4，不是 4.5！

double g = 9.0 / 2;
// 结果是 4.5 (只要有一个是浮点数，结果就是浮点数)
```

### 坑点 B：取模运算 (Modulo)
> 符号：% (取余数)
> 
> 用途：判断奇偶数 (n%2)、限制范围。

```Java
int e = 21 % 4; 
// 结果是 1 (21除以4商5，余1)
```

## 3. 魔鬼细节：自增与自减 (++/--) 

这是本节最难、最易错、考试必考的知识点！请务必盯着看三遍。

> 基本原理：让变量的值加 1 或减 1。
> 
> 区别：<span style="background:rgba(240, 107, 5, 0.2)">符号写在前面还是后面</span>，决定了是“先干活”还是“先拿钱”。

### ⚔️ 前置 vs 后置 (Pre vs Post)

| **类型**   | **写法** | **口诀**   | **逻辑详解**                                    |
| -------- | ------ | -------- | ------------------------------------------- |
| **前置自增** | `++a`  | **先变再用** | 1. 变量自己先加 1。<br>2. 然后把**新值**拿去参与运算。         |
| **后置自增** | `a++`  | **先用再变** | 1. 先把变量**原来的值**拿去参与运算。<br>2. 运算完后，变量自己再加 1。 |
### 考试防坑代码 (必背)

```Java
// 场景 1：后置 a++ (无私奉献型)
int a = 1;
int b = a++; 
// 过程：
// 1. 先把 a(1) 的值赋给 b -> 所以 b 是 1
// 2. a 自己再加 1 -> 所以 a 变成了 2
System.out.println(a); // 2
System.out.println(b); // 1

// 场景 2：前置 ++c (先己后人型)
int c = 1;
int d = ++c;
// 过程：
// 1. c 自己先加 1 -> 所以 c 变成了 2
// 2. 把新值 c(2) 赋给 d -> 所以 d 是 2
System.out.println(c); // 2
System.out.println(d); // 2
```
# 课后习题

### 1. 自增自减 (Increment/Decrement)

**题目 1**：`int a=10, b=4, c=20, d=6`，求 `a++ * b + c * --d`

- **答案**：**140 (C)**
    
- **教授解析**：
    
    - `a++` (后置)：<span style="background:rgba(236, 57, 57, 0.55)">先用 10 算，a 变 11</span>。
        
    - `--d` (前置)：<span style="background:rgba(240, 167, 216, 0.55)">d 先变 5，用 5 算</span>。
        
    - 算式：`10 * 4 + 20 * 5` = `40 + 100` = `140`。

**题目 2**：`a=2, b=2`，下列值不为 4 的是？

- (A) `a * (++b)`
    
- (B) `a * (b++)`
    
- **答案**：**A**
    
- **解析**：
    
    - A: `++b` 先变 3，`2 * 3 = 6` (不是4)。
        
    - B: `b++` 先用 2，`2 * 2 = 4`。
        

**题目 3 (连环坑)**：求程序运行结果

```Java
float i = 3.0f;
int n = 10;
System.out.print(i++); 
System.out.println("," + n * i++);
```

- **答案**：**B (3.0, 40.0)**
    
- **教授解析**：
    
    1. 第一次 `i++`：后置，先打印 **3.0**。内存中 `i` 变为 4.0。
        
    2. 第二次 `i++`：后置，先用 **4.0** 乘 10 $\rightarrow$ 40.0。
        
    3. 结果拼接：`3.0,40.0`。

### 8. 表达式结果类型判断

已知 `int a=5; int b=2;`

|**表达式**|**结果**|**类型**|**原因**|
|---|---|---|---|
|`a/b`|**2**|`int`|整数除法，截断小数。|
|`a%b`|**1**|`int`|5 除以 2 余 1。|
|`(float)a/b`|**2.5**|`float`|`a` 先变成了 5.0，浮点除以整数=浮点。|
|`(float)(a/b)`|**2.0**|`float`|先算括号里 `5/2=2`，再把 2 转成 float。|

## 第四部分：编程大题实战 (Programming) 

### 9. 水仙花数/拆分数字 (Digit Split)

**题目**：从控制台输入输入一个三位数（如 123），分别输出个、十、百位，并求出个十百位的和。

**满分代码模板 (直接背)**

```Java
import java.util.Scanner; // 1. 导包

public class DigitDemo {
    public static void main(String[] args) {
        // 2. 准备输入
        Scanner sc = new Scanner(System.in);
        System.out.println("请输入一个三位数：");
        int n = sc.nextInt(); // 假设输入 123
        
        // 3. 核心算法 (必背!)
        int bai = n / 100;        // 123 / 100 = 1
        int shi = n / 10 % 10;    // 123 / 10 = 12 -> 12 % 10 = 2
        int ge  = n % 10;         // 123 % 10 = 3
        
        // 4. 输出
        System.out.println("百位：" + bai);
        System.out.println("十位：" + shi);
        System.out.println("个位：" + ge);
        System.out.println("总和：" + (bai + shi + ge));
    }
}
```

> [!TIP] **记忆口诀**： 
> - 想拿最右边的数（个位） $\rightarrow$ **模 10 (`% 10`)**
> - 想干掉最右边的数 $\rightarrow$ **除 10 (`/ 10`)**



###  重点考点：负数取模规则
> **公式**：`a % b`
> **铁律**：结果的符号 **永远与 a (左边的数) 保持一致**！
* `10 % -3` $\rightarrow$ 看左边 `10` 是正 $\rightarrow$ 结果 `1`
* `-10 % 3` $\rightarrow$ 看左边 `-10` 是负 $\rightarrow$ 结果 `-1`

>[!WARNING] 在 Java（以及 C/C++）中，取模运算 `a % b` 的结果的正负号，**只和 `a`（被除数/左边的数）的符号有关**，完全不用管 `b`（除数/右边的数）是正还是负！
