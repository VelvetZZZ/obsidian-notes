# For 循环与作用域

## 1. For 循环的解剖 (Anatomy) 🔬
> **应用场景**：当你明确知道要循环多少次时（比如跑 10 圈、打印 100 次），`for` 是首选。

### A. 语法结构
```java
//   (1)初始化    (2)门禁条件   (4)步进/迭代
for (int i = 0;   i < 5;      i++) {
    // (3) 循环体 (干活)
    System.out.println(i);
}
// 执行顺序：1 -> 2 -> 3 -> 4 -> 2 -> 3 -> 4 ... -> 2(False) -> 结束
```
### B. For vs While 的“变形计” (必考转换) 🔄

PPT 专门对比了这两种写法，核心区别在于 **变量 `i` 活得久不久**。

| **特性**    | **While 循环**                               | **For 循环**                                       |
| --------- | ------------------------------------------ | ------------------------------------------------ |
| **代码结构**  | 散装 (初始化在外面，迭代在里面)                          | **整装** (一行代码全搞定)                                 |
| **变量作用域** | **大** (初始化变量在循环外，属于==全局变量==，循环结束后 `i` 还能用) | **小** (初始化变量在 `()` 里，是==局部变量==，**出了循环 `i` 就死了**) |

###  变量作用域陷阱 (Scope Trap)

```Java
// while 写法
int i = 0;
while(i < 5) { i++; }
System.out.println(i); // ✅ 正确，i 还在内存里

// for 写法
for(int j = 0; j < 5; j++) { ... }
System.out.println(j); // ❌ 报错！j 已经“死”在循环里了，外面不认识它。
```

## 2. 只有 For 才能做的骚操作 🕺

PPT 里的习题展示了 `for` 的灵活性：

1. **无限循环**：<span style="background:rgba(236, 57, 57, 0.55)">for(;;) 等同于 while(true)</span> 。
    
2. **多变量控制**：`for(int x=0, y=0; x<5; x++, y--)` —— 你可以在括号里写一堆变量，只要用<span style="background:#fdbfff">逗号</span>隔开就行。
    
3. **省略**：初始化和步进都可以省略（只要分号留着），**但千万<span style="background:rgba(236, 57, 57, 0.55)">别省略条件</span>，否则死循环**。

### A. 阶乘 (Factorial)

- **逻辑**：$1 \times 2 \times 3 \dots \times n$。
    
- **注意**：累乘器 `result` 初始值必须是 **1** (如果写 0，乘完还是 0)。

### B. 水仙花数 (Narcissistic Number)

- **逻辑**：`153` -> $1^3 + 5^3 + 3^3 = 1 + 125 + 27 = 153$。
    
- **核心技巧 (拆数字)**：
    
    - 个位：`n % 10`
        
    - 十位：`n / 10 % 10`
        
    - 百位：`n / 100`

###  习题深度复盘 (揪出你的逻辑漏洞)

#### **❌ 第 1 题：For 循环的执行顺序 (选 C -> 正解 D)**


**你的误区**：你可能以为是求 1+2+3+4+5=15。
**致命细节**：注意 `i++` 的位置！它在**循环体里面**，而且在**累加之前**！

**🔍 教授带你跑代码 (Trace)：**
```java
int sum = 0;
int i = 1;
for (; i <= 5; ) { // 只要 i <= 5 就进门
    i++;           // 1. 进门先自增！(i 变成 2)
    sum += i;      // 2. 然后加 i (sum = 0 + 2 = 2)
}
````

- **第 1 轮**：`i`=1。满足。进门。`i`变2。`sum`+2=**2**。
    
- **第 2 轮**：`i`=2。满足。进门。`i`变3。`sum`+3=**5**。
    
- **第 3 轮**：`i`=3。满足。进门。`i`变4。`sum`+4=**9**。
    
- **第 4 轮**：`i`=4。满足。进门。`i`变5。`sum`+5=**14**。
    
- **第 5 轮**：`i`=5。**满足！** 进门。**`i`变6**。`sum`+6=**20**。
    
- **第 6 轮**：`i`=6。不满足。退出。
    
- **结果**：**20** (选 D)。

#### **❌ 第 2 题：逻辑短路 (选 B -> 正解 D)**

题目：for(int x=0, y=0; (y!=0) && (x<4); x++)

你的误区：你可能只看到了 x<4，觉得会跑 4 次。

致命细节：(y!=0)。

**🔍 教授逻辑分析：**

1. **初始化**：`x=0`, `y=0`。
    
2. **判断条件**：`(y!=0) && (x<4)`
    
    - `y` 是 0。
        
    - 所以 `y!=0` 是 **False**。
        
    - **短路**：前面是 False，`&&` 后面不管是什么，整体都是 **False**。
        
3. **结果**：门禁第一关就被拦下了，循环体 **一次都没执行**。
    
4. **答案**：一次也不执行 (选 D)。_(注：这也解释了为什么第 5 题你填 0 是对的，因为是同一道题)_

### 💻 编程题速解 (满分模板)

#### **题 1：求阶乘 (Factorial)**

> 输入一个小于 10 的整数 n，求 n!

Java

```
import java.util.Scanner;
public class Factorial {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        System.out.println("请输入一个整数n (n<10):");
        int n = sc.nextInt();
        
        long result = 1; // 乘法初始值要是1，防止溢出用long
        for (int i = 1; i <= n; i++) {
            result *= i; // 等价于 result = result * i
        }
        System.out.println(n + "的阶乘是：" + result);
    }
}
```

#### **题 2：特殊的四位数**

> 满足 (ab+cd)*(ab+cd) = abcd。
> 
> 技巧：不要拆千位百位那么麻烦，直接用 /100 拿前两位，%100 拿后两位。


```Java
public class SpecialNumber {
    public static void main(String[] args) {
        // 遍历所有四位数
        for (int n = 1000; n <= 9999; n++) {
            int left = n / 100;  // 前两位 (比如 3025 -> 30)
            int right = n % 100; // 后两位 (比如 3025 -> 25)
            
            int sum = left + right;
            
            // 判断平方是否等于原数
            if (sum * sum == n) {
                System.out.println("找到啦：" + n);
            }
        }
    }
}
```

#### **题 3：韩信点兵 (Han Xin Counting)**

> 题目：
> 
> 不足百人 ($<100$)。
> 
> 3人一行 多1人 (余1)。
> 
> 7人一行 少2人 (注意：7人少2人 = 5人，即 余5)。
> 
> 5人一行 正好 (余0)。

```Java
public class HanXin {
    public static void main(String[] args) {
        // 人数不足100，所以从 1 遍历到 99
        for (int i = 1; i < 100; i++) {
            // 条件翻译：
            // 1. i % 3 == 1 (多1)
            // 2. i % 7 == 5 (少2意味着还差2个就满7了，说明现在有5个 -> 余5)
            // 3. i % 5 == 0 (正好)
            if (i % 3 == 1 && i % 7 == 5 && i % 5 == 0) {
                System.out.println("韩信带了 " + i + " 人");
            }
        }
    }
}
```
