# While 循环语句

## 1. 核心结构 (Structure) 🔄
> **口诀**：**先判断，后执行**。(可能一次都不执行)

```java
int i = 1; // 1. 初始化 (起点)
while (i <= 100) { // 2. 循环条件 (终点)
    System.out.println(i); // 3. 循环体 (干活)
    i++; // 4. 迭代 (步进) -> ⚠️ 忘了写这行就是死循环！
}
```
## 2. While vs Do-While (必考区别)

- **`while`**：先检查门票。如果条件一开始是 false，循环体**一次都不执行**。
    
- **`do-while`**：先斩后奏。无论条件如何，循环体**至少执行一次**。
## 3. 编程大题实战策略 

- **已知循环次数**（如：计算 1-100 的和）：用 `for` 或 `while` 都可以，`for` 更简洁。
    
- **未知循环次数**（如：一直输入直到输入正确密码）：必须用 `while`。


### 💻 教授的引导式编程题 (Guided Challenge)

根据你的要求，我不给答案，我给你**题目**和**思路导航**。我们来做 PPT 第 11 页 的 **编程题 2**。

**题目：**
> 从控制台输入一个正整数 $n$，计算 $1 + 1/2 + 1/3 + ... + 1/n$ 的结果。

**🤔 思考陷阱 (做对这题的关键)：**
Java 里的整数除法 `1/2` 等于多少？是 `0.5` 吗？不，是 `0`！
所以，如果你写 `sum = sum + 1/i;`，你算出来的永远是 1。怎么破？

**🚀 教授导航 (请在纸上写代码)：**

1.  **准备工作**：
    * 你需要 `Scanner` 来接收 $n$。
    * 你需要一个变量 `sum` 来存结果。**注意：结果有小数，`sum` 应该是什么类型？** (`int` 还是 `double`?)
2.  **循环设计**：
    * 你需要一个变量 `i` 代表分母，从 1 开始。
    * `while` 的条件是 `i` 小于等于 $n$。
3.  **核心公式 (最关键的一步)**：
    * `sum = sum + 1.0 / i;` (必须把 1 写成 `1.0`，强行让它变成浮点运算！)
    * 别忘了 `i++`。
