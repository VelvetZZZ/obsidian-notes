# ☕ Java期末速通：循环嵌套 (套娃逻辑)

## 1. 核心概念 (The Concept) 
> **定义**：循环里面套循环。
> **形象比喻**：
> * **外层循环**：时钟的 **时针** (转得慢，控制行数)。
> * **内层循环**：时钟的 **分针** (转得快，控制每一行里的内容)。

### 执行流程
* 外层走 **1** 步，内层要跑 **一整圈**！
* 假设外层 `i` 跑 5 次，内层 `j` 跑 10 次，总共执行 $5 \times 10 = 50$ 次。
---

## 2. 作用域陷阱 (The Scope Trap - P5 错误详解) 
你提到的 **PPT 第 5 页** 的错误非常经典，是期末代码改错题的常客。

### ❌ 错误代码还原
```java
for (int i = 1; i <= 4; i++) {
    int j = 1; // <--- 变量 j 在外层循环里“出生”了
    
    // 第一个内层循环
    for (; j <= 4 - i; j++) { ... } 
    
    // ❌ 第二个内层循环 (报错点!)
    for (int j = 1; j <= i; j++) { ... } 
}
```
### 教授解析

- **报错原因**：**"Variable 'j' is already defined"** (变量 j 已定义)。
    
- **原理**：`int j` 是在 `for(i)` 的大括号里定义的。只要没出这个大括号，`j` 就一直活着。
    
- **冲突**：你在下面又写了一句 `int j = 1`，相当于在同一个屋子里想生两个叫 "j" 的孩子，Java 不允许！
    
- **✅ 修正方法**：第二个循环里的变量改名叫 `k`，或者直接用 `j` (<font color="#ff0000">去掉 `int` 声明</font>)。

### 变量访问规则

- **内层** 可以用 **外层** 的变量 (子承父业)。
    
- **外层** **不能** 用 **内层** 的变量 (出了内层循环，内层变量就“死”了)。

## 3. 必背经典案例 (Classic Patterns) 

### A. 九九乘法表 (Multiplication Table)

> **逻辑核心**：第 `i` 行有 `i` 列。 **控制条件**：内层循环 `j <= i`。

```Java
for (int i = 1; i <= 9; i++) {     // 行
    for (int j = 1; j <= i; j++) { // 列：不能超过行号
        System.out.print(j + "*" + i + "=" + (i*j) + "\t");
    }
    System.out.println(); // 换行
}
```

### B. 百钱买百鸡 (Brute Force)

> **题目**：公鸡 5 文，母鸡 3 文，小鸡 3 只 1 文。100 文买 100 只。 **技巧**：直接三层 (或两层) 循环暴力穷举。

```Java
// 优化版：两层循环 (小鸡数量 = 100 - 公鸡 - 母鸡)
for (int x = 0; x <= 20; x++) { // 公鸡最多 20
    for (int y = 0; y <= 33; y++) { // 母鸡最多 33
        int z = 100 - x - y; // 小鸡
        // 两个条件：1.总钱数100  2.小鸡必须是3的倍数(否则除不尽)
        if (z % 3 == 0 && (5*x + 3*y + z/3 == 100)) {
            System.out.println("公鸡:" + x + " 母鸡:" + y + " 小鸡:" + z);
        }
    }
}
```
### 习题 1 深度复盘 (选 C -> 正解 A)

**题目图示**：
```java
for (int i = 2; i >= 0; i--) { // i: 2, 1, 0
    for (int j = 0; j < 2 * i + 1; j++) { // 打印星星
        System.out.print("*");
    }
    System.out.println(); // 换行
}
````

**🕵️‍♀️ 教授逻辑推演**：

1. **第一轮 (`i=2`)**：`j < 2*2+1 = 5`。打印 **5 个星** `*****`。
    
2. **第二轮 (`i=1`)**：`j < 2*1+1 = 3`。打印 **3 个星** `***`。
    
3. **第三轮 (`i=0`)**：`j < 2*0+1 = 1`。打印 **1 个星** `*`。
    

**为什么选 A 不选 C？**

- **A 选项**：`*****` (靠左对齐)。
    
- **C 选项**：`*****` (居中对齐，倒金字塔)。
    
- **关键点**：代码里**只有打印 `*` 的语句，没有打印空格 (`" "`) 的语句**！没有空格怎么可能居中呢？它只能乖乖靠左边站。
    
- **结论**：**选 A**。

### 💻 编程题速解 (Pyramid Patterns)

题目要求打印 **金字塔 (Pyramid)** 和 **倒金字塔**。这需要 **“先打空格，再打星星”** 的混合双打。

#### **题 1：正金字塔 (Upright Pyramid)**

> 规律：第 `i` 行 (从1开始)，空格数 `n-i`，星星数 `2*i-1`。

```Java
public class Pyramid {
    public static void main(String[] args) {
        int n = 5; // 总行数
        
        for (int i = 1; i <= n; i++) {
            // 1. 先打印空格 (左边的空白)
            for (int j = 1; j <= n - i; j++) {
                System.out.print(" ");
            }
            // 2. 再打印星星
            for (int k = 1; k <= 2 * i - 1; k++) {
                System.out.print("*");
            }
            // 3. 换行
            System.out.println();
        }
    }
}
```

#### **题 2：倒金字塔 (Inverted Pyramid)**

> 规律：第 `i` 行 (我们让 i 从 n 倒着数到 1)，逻辑和上面一模一样！


```Java
public class InvertedPyramid {
    public static void main(String[] args) {
        int n = 5; // 总行数
        
        // 技巧：i 直接从 5 减到 1，里面的逻辑完全不用变！
        // 这样第一行就是 i=5 的情况 (0空格，9星)
        for (int i = n; i >= 1; i--) {
            
            // 1. 打印空格
            for (int j = 1; j <= n - i; j++) {
                System.out.print(" ");
            }
            // 2. 打印星星
            for (int k = 1; k <= 2 * i - 1; k++) {
                System.out.print("*");
            }
            // 3. 换行
            System.out.println();
        }
    }
}
```
