# 标识符与命名规范 

## 1. 标识符规则 (硬性语法) 
> **定义**：用来给类、方法、变量起名字的字符序列。
> **注意**：如果不遵守这些规则，代码会直接报错（编译不通过）。

* **组成元素**：
    * ✅ 字母 (A-Z, a-z)
    * ✅ 数字 (0-9)
    * ✅ 下划线 (`_`)
    * ✅ 美元符号 (`$`)
* **三大禁忌** (考试常考判断/纠错)：
    1.  ❌ **不能以数字开头**。
    2.  ❌ **不能包含空格**。
    3.  ❌ **不能是 Java 关键字或保留字** (如 `class`, `int`, `public` 等)。
### 考试防坑实例
```java
int 2days = 10;     // ❌ 错误：数字开头
int my age = 20;    // ❌ 错误：包含空格
int class = 5;      // ❌ 错误：class是关键字
int $salary = 100;  // ✅ 正确：虽然不推荐用$，但在语法上是允许的
int _count = 0;     // ✅ 正确：下划线开头允许
```
## 2. 命名规范 (软性约定) 

> 定义：程序员之间的“潜规则”。
> 注意：不遵守代码能跑，但会被扣分（尤其手写代码题），且显得不专业。

| **类型**             | **规范名称**                                                          | **规则描述**                            | **实例 (Example)**                    |
| ------------------ | ----------------------------------------------------------------- | ----------------------------------- | ----------------------------------- |
| **类名 (Class)**     | <span style="background:rgba(236, 57, 57, 0.55)">大驼峰(PascalCase)  | **每**个单词首字母大写                       | `HelloWorld`, `StudentInfo`         |
| **方法名 (Method)**   | <span style="background:rgba(240, 167, 216, 0.55)">小驼峰(camelCase) | 第1个单词</span>首字母小写，后面单词首字母大写,不包含$和_s | `getAge()`, `calculateTotalPrice()` |
| **变量名 (Variable)** | <span style="background:rgba(240, 167, 216, 0.55)">小驼峰(camelCase) | 同方法名</span>                         | `studentName`, `totalScore`         |
| **常量 (Constant)**  | <span style="background:#ff4d4f">全大写(Snake Case) </span>          | 全部大写，单词间用下划线 `_` 分隔                 | `MAX_VALUE`, `PI`                   |
| **包名 (Package)**   | <span style="background:rgba(240, 107, 5, 0.2)">全小写</span>        | 全部小写，通常用域名反写                        | `com.google.search`                 |

### 考试防坑实例

- **手写代码题常见扣分项**：
    
    - 类名写成 `helloworld` (❌ 每个单词首字母未大写)
        
    - 变量写成 `StudentName` (❌ 首字母大写了，像类名)
        
    - 常量写成 `maxvalue` (❌ 没大写且没下划线)
    
- **特别提醒**：尽量不要用拼音！`int nianling` (❌ LOW) vs `int age` (✅ Pro)。

## 3. 关键字 (Keywords) 🔑

> Java中赋予了特殊含义的单词，**全都是小写**。

- **常见关键字**：`public`, `static`, `void`, `class`, `int`, `if`, `else`, `return` 等。
    
- **冷门考点 (保留字)**：
    
    - `goto` 和 `const`：虽然Java现在不用它们，但它们是**保留字**，也不能用来做变量名！
        
    - `true`, `false`, `null`：它们严格来说是**字面量 (Literals)**，不是关键字，但也不能用来做标识符。

```java
int goto = 10; // ❌ 编译错误，goto是保留字
int true = 1;  // ❌ 编译错误
```
