#### 考前救急：数组 vs 链表 (必考对比题)

期末考试最喜欢问你：_“在这个场景下，选数组还是选链表？”_ 背下这个对比表：

| **特性**    | **数组 (Array) / Python List** | **单链表 (Linked List)**    | **记忆口诀**         |
| --------- | ---------------------------- | ------------------------ | ---------------- |
| **访问**    | **快** $O(1)$                 | **慢** $O(n)$             | 数组是直达电梯，链表是爬楼梯   |
| **插入/删除** | **慢** $O(n)$ (要搬家移动后面所有元素)   | **快** $O(1)$ (只要改一下指针指向) | 链表==加塞==容易，数组加塞难 |
| **空间**    | 必须连续的一大块内存                   | 散落在内存各处，用指针连起来           | 数组是整租房，链表是散租房    |
| **长度**    | 固定 (传统数组)                    | 动态 (想加就加)                |                  |
