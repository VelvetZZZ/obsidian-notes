# 编程题
6-1 堆排序

实现堆排序，函数void HeapAdjust(SqList &L,int s,int m)为筛选法调整堆，函数void CreatHeap(SqList &L)把无序序列L.r[1..n]建成大根堆，函数void HeapSort(SqList &L)对顺序表L进行堆排序。

### 函数接口定义：

```c++
void HeapAdjust(SqList &L,int s,int m);//筛选法调整堆
void CreatHeap(SqList &L); //把无序序列L.r[1..n]建成大根堆
void HeapSort(SqList &L);//对顺序表L进行堆排序
```

其中 `L` 为顺序表，函数HeapAdjust假设`L.r[s+1..m]` 已经是堆，将`L.r [s..m]` 调整为以`L.r[s]` 为根的大根堆。

void HeapAdjust(SqList &L, int s, int m) {

// 假设 L.r[s+1..m] 已经是堆，将 L.r[s..m] 调整为以 s 为根的大根堆

ElemType temp = L.r[s];

for (int j = 2 * s; j <= m; j *= 2) {

// 选出左右孩子中 key 更大的那个

if (j < m && L.r[j].key < L.r[j + 1].key) j++;

// 若父结点已不小于最大孩子，结束

if (temp.key >= L.r[j].key) break;

// 否则孩子上移

L.r[s] = L.r[j];

s = j;

}

L.r[s] = temp;

}

  

void CreatHeap(SqList &L) {

// 建大根堆：从最后一个非叶子结点开始向前调整

for (int i = L.length / 2; i >= 1; --i) {

HeapAdjust(L, i, L.length);

}

}

  

void HeapSort(SqList &L) {

// 升序：先建大根堆，再不断把堆顶(最大值)交换到末尾

CreatHeap(L);

for (int i = L.length; i > 1; --i) {

ElemType tmp = L.r[1];

L.r[1] = L.r[i];

L.r[i] = tmp;

HeapAdjust(L, 1, i - 1);

}
