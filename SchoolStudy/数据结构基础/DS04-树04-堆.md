## 6-1 Percolate Up and Down

Score 20

Author 陈越

Organization 浙江大学

Write the routines to do a "percolate up" and a "percolate down" in a binary min-heap.

### Format of functions:

```c++
void PercolateUp( int p, PriorityQueue H );
void PercolateDown( int p, PriorityQueue H );
```

where `int p` is the position of the element, and `PriorityQueue` is defined as the following:

```c++
typedef struct HeapStruct *PriorityQueue;
struct HeapStruct {
    ElementType  *Elements;
    int Capacity;
    int Size;
};
```

### Sample program of judge:

```c++
#include <stdio.h>

typedef int ElementType;

struct HeapStruct {
    ElementType  *Elements;

void PercolateUp( int p, PriorityQueue H );
void PercolateDown( int p, PriorityQueue H );

    int p = ++H->Size;

    PercolateUp( p, H );

    ElementType MinElement;
```

void PercolateUp( int p, PriorityQueue H ) {

ElementType temp = H->Elements[p]; // 保存当前要调整的元素

// 从p开始向上遍历，直到根节点（p=1）或父节点更小

for (; p > 1 && temp < H->Elements[p/2]; p /= 2) {

H->Elements[p] = H->Elements[p/2]; // 父节点下移

}

H->Elements[p] = temp; // 找到合适位置，放入当前元素

}

  

void PercolateDown( int p, PriorityQueue H ) {

ElementType temp = H->Elements[p]; // 保存当前要调整的元素

int child; // 用于记录最小子节点的下标

// 循环直到当前节点是叶子节点（没有左子节点）

for (; 2*p <= H->Size; p = child) {

child = 2 * p; // 左子节点下标

// 找到左右子节点中值更小的那个（注意右子节点可能不存在）

if (child != H->Size && H->Elements[child+1] < H->Elements[child]) {

child++; // 右子节点更小，切换到右子节点

}

// 如果当前元素大于最小子节点，交换位置；否则无需继续调整

if (temp > H->Elements[child]) {

H->Elements[p] = H->Elements[child]; // 子节点上移

} else {

break; // 找到合适位置，退出循环

}

}

H->Elements[p] = temp; // 放入当前元素

}
